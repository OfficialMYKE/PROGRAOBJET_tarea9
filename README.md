
# 1. Sistema de Juegos

### Descripción General  
Este módulo implementa un conjunto de clases para representar diferentes tipos de videojuegos.  
La clase `Juego` sirve como estructura base, mientras que las clases hijas implementan características específicas.

### Funcionalidades  
- Clase abstracta base (`Juego`)  
- Métodos abstractos sobrescritos por clases hijas  
- Encapsulamiento de atributos  
- Implementación orientada a objetos  
- Facilidad para agregar nuevos tipos de juegos

# 2. Sistema de Dispositivos Electrónicos

### Descripción General

Este sistema gestiona diferentes dispositivos electrónicos mediante una jerarquía de clases.
Incluye validaciones, manejo de errores y técnicas básicas de polimorfismo.

### Funcionalidades

* Herencia entre dispositivos
* Validación de datos en constructores
* Manejo de excepciones mediante `try-catch`
* Métodos específicos para cada tipo de dispositivo
* Arquitectura clara y extensible
